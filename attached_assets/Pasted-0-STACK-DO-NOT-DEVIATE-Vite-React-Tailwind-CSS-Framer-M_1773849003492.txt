9. WHAT TO BE CAREFUL ABOUT

If Replit drifts, it will:

Add too much spacing

Use wrong colors (too bright or too gray)

Over-round everything

Use generic charts

Force:

Tight spacing

Dark, rich tones

Subtle, not loud UI
